# GILSON PASSOS CONSTRUTORA — Sistema Financeiro

Sistema Web de Controle Financeiro e Operacional.

## Funcionalidades

### Módulo Empresarial
- Dashboard com gráficos em tempo real
- Gestão de Obras/Projetos
- Controle de Recebimentos por entidade (MEI, PF, Espécie)
- Limites Fiscais: MEI (R$ 81k/ano) e PF (R$ 5k/mês) com alertas visuais
- Caixa em Espécie com depósito manual para bancos
- Gestão de Bancos vinculados às entidades
- Cartão de Ponto para diaristas
- Folha Semanal de pagamentos
- Registro de Despesas corporativas

### Módulo Pessoal
- Dashboard de saúde financeira da casa
- Gastos Fixos (aluguel, energia, etc.)
- Gastos Variáveis (mercado, lazer, etc.)
- Orçamento mensal com saldo livre

## Como Usar

1. Abra o arquivo `index.html` no navegador (ou acesse o link online)
2. Na primeira vez, clique em "Configurar acesso" e crie sua conta (e-mail + senha)
3. Faça login e navegue pelo menu lateral

## Deploy no Netlify (Gratuito)

1. Crie uma conta em https://netlify.com
2. Arraste a pasta do projeto para o painel do Netlify (Drag & Drop)
3. Seu site estará online em segundos com URL própria

## Dados

Todos os dados ficam salvos no navegador (localStorage). Para backup:
- Use a função de exportar dados (futura implementação)
- Não limpe o histórico/dados do navegador para não perder informações
